# KD Sarees – Frontend project

React (Vite) client for KD Sarees.

## Structure

```
kd_sarees/
├── client/            # React (Vite) – frontend
│   └── src/
│       ├── components/   # Reusable UI
│       ├── pages/        # Page/screen components
│       ├── services/     # External API clients (when needed)
│       ├── hooks/        # Custom React hooks
│       ├── context/      # React Context providers
│       ├── assets/       # Images, fonts
│       ├── styles/       # Global/shared styles
│       └── utils/        # Helpers
```

## Run

```bash
cd client
npm install
npm run dev
```

See `client/README.md` and folder `README.md` files for details.
